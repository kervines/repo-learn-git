Git & GitHub

Guia para auxiliar no estudo de Git e GitHub

Comandos Básicos Git

  • Para iniciar o git no projeto:

    git init

  • Para verificar o status do projeto, modificações e adição de arquivos:

    git status

  • Para preparar o arquivo para o commit:

    git add (nome-arquivo)

    Caso seja adicionado mais de um arquivo:

    git add .

  • Para realizar o commit e salvar as alterações:

    git commit -m "informação do arquivo adicionado"

    Caso a alteração seja realizado dentro do arquivo

    git commit -a -m "informação da alteração realizado no arquivo"

Comandos

  • Para evitar as alterações para o repositório em nuvem

    git push origin (nome-branch)

    • Branch support

    Comando para adicionar uma branch:

    git checkout (nome-branch)

    Comando para adicionar uma branch e mudar para a nova branch

    git checkout -b (nome-branch)
